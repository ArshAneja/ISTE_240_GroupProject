Data Types

Lists

A collection of things. The list can include strings, integers, floats, booleans, etc. A list is denoted by brackets []

Tuples

Is also a collection of things. However, there is a fixed length. Once the tuple has been created, it cannot be changed. The tuple can include strings, integers, floats, booleans, etc.

Sets

a data structure that guarantees that all of the elements that it contains are unique. The list can include strings, integers, floats, booleans, etc. It is denoted with {}. If an element is repeated, it will not be printed.

Dictionaries

A data structure where a key can be used to retrieve a value from the dictionary. The key can be a string, integer, floats, booleans, ets and the value can be anything as well. The Python dictionary is a hashing data structure. It is denoted with {} or ().

Arrays

A data structure that has a fixed length. Once the array is created, the length of the array never changes. Once the memory is allocated, it is filled with some prototype value (none, or zero). Python does not support arrays, so you must import it. All elements of the array must be the same type.
